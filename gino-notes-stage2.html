Important Concepts - Telling Computers What to Do

Lesson 3: Introduction to "Serious" Programming

Computer

Most machines (like a toaster) are designed to do a few things. Unless you physically alter these machines, they will only be able to do those things.

Computers are different. A computer can be programmed to do anything we want, as long as we can write a program that specifies a specific sequence of instructions.

Computer Program

A program is a precise sequence of steps that a computer can follow to do something useful. Web browsers, games, mobile apps, and simple print statements are all examples of computer programs.

Programming Language

A programming language is what programmers utilize to instruct a computer what to do. Python is one example of a programming language.

What is a compiler?
A compiler is a special program that processes statements written in a particular programming language and turns them into machine language or "code" that a computer's processor uses. An example of a compiled language is Visual Basic.
What is an Interpreter?
An interpreter checks the errors of a program statement by statement. An example of an interpreted language is python.
What Are Debuggers?

Debuggers are computer programs that assist (mostly programmers) in the detection and correction of errors in computer programs. The term debugging is the process of locating and fixing or bypassing bugs (errors) in computer program code.

Python Language

When you write Python code and press "Run", a Python Interpreter converts the code you wrote as a set of instructions that the computer itself can understand and execute.

Grammar

Just like human languages have grammars, programming languages do too. A grammar is a specification of what is "correct" and what is "incorrect."

In languages like English, people can ofent make sense of sentences that aren't technically "correct," but computers aren't smart enough to do this. This means we have to write code that is exactly "correct" according to the Python interpreter, otherwise our code won't run.

Python Expressions

A Python "expression" is a legal Python statement. For example: print 1 + 1 is a valid expression, but print 1 + (without a number at the end) is not.

This video does a good job of explaining the Python grammar for "expressions".

What is a variable and how to use it?

A variable is a temporary storage for data. You can use variables to store all sorts of data in Python or any other programming language. The name of the variable is always on the left side of the equals sign (the assignment operator), and the value of the variable on the right side of the equals sign. First the name, then the value. In other languages, you need to specify the type (e.g. string, numeric - integer, long integer, etc, date) of variable you want to use before you can store data in it. That's not the case in Python however, it is a good programming habit to use comments to document the type of variable you are using. In addition, you can name your variables according to the data type. Example: if you want to store integer then it makes sense to name your variable intVariable.

The difference between the equals sign in the syntax of assigning a variable and the equals sign after an arithmetic calculation is that the equals sign in the latter represents the output of a result. There is no storage associated with it whereas in the former, it represents the input i.e. you are assigning a new value to a variable.

Variables are useful in mathematical calculations, reusability of data in the program, helps to organize code better.

IF...ELIF...ELSE Statements

The if statement is a conditional that, when it is satisfied, activates some part of code. Syntax: if expression: statement(s) else: statement(s)

Note: Python does not provide switch or case statements as in other languages

Function

A function is a block of cohesive, reusable code that is used to perform a single, related action. Functions provide better modularity and a high degree of code reusing.

Difference between making and using a function

Every function is made by writing the line of code with the keyword def followed by the function name along with a list of parameters (or arguments if you like) enclosed by parentheses and a colon. The number of arguments defined in the function should be equal to the number of values passed in to a function when it is called out in the application.

In the body of the function we write the code which can be one line or a block of code that specified what to do with the parameters. The return statement is important in the function to indicate what kind of value will be produced as the output. Without the return statement your function will output nothing but code will still be performed inside the function.

Example:

def display_greeting(strname):
return "Hello " + strname

print display_greeting("Johnny")
>>>Hello Johnny

Example of a function without the return statement

def display_greeting(strname):
strname = "Hello " + strname

print display_greeting("Johnny")
>>>None
